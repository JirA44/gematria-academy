Exercice #1
⭐ Facile

Déchiffrez ce mot hiéroglyphique

𓈖𓆑𓂋

Consigne : Identifiez chaque signe et écrivez la prononciation complète (lettres latines)

💡 Indice :
• Le premier signe (zigzag) = N (eau)
• Le deuxième signe (serpent) = F (vipère)
• Le troisième signe (bouche) = R
Exercice #2
⭐ Facile

Déchiffrez avec déterminatif

𓇋𓏠𓈖𓀭

Consigne : Quelle est la prononciation ? (Attention, le dernier signe est un déterminatif muet !)

💡 Indice :
• 𓇋 = I (roseau)
• 𓏠 = M (chouette variante)
• 𓈖 = N (eau)
• 𓀭 = déterminatif "dieu" (ne se prononce PAS)
Exercice #3
⭐⭐ Moyen

Phrase courte

𓅱𓈖 𓈖𓆑𓂋

Consigne : Cette phrase signifie "être bon" ou "il est bon". Quelle est la prononciation complète ?

💡 Indice :
• Mot 1: 𓅱𓈖 = WN (être, exister)
• Mot 2: 𓈖𓆑𓂋 = NFR (bon, beau)
• Prononciation: WN NFR (ou "ounen nefer")
Exercice #4
⭐⭐ Moyen

Cartouche royal

𓇳𓏤𓊃𓇓

Consigne : Identifiez les signes de ce nom royal. Quelle est la prononciation ?

💡 Indice :
• 𓇳 = RA (le soleil)
• 𓏤 = 1 (trait unique)
• 𓊃 = S (verrou)
• 𓇓 = déterminatif royal
• Ce nom se lit: RA-S ou RAMSÈS (Ramsès)
Exercice #5
⭐⭐⭐ Difficile

Formule d'offrande

𓎛𓏏𓊪 𓂧𓇋 𓈖𓋴𓅱𓏏

Consigne : Formule très fréquente signifiant "offrande que donne le roi". Déchiffrez la prononciation.

💡 Indice :
• 𓎛𓏏𓊪 = HTP (offrande, autel)
• 𓂧𓇋 = DI (donner)
• 𓈖𓋴𓅱𓏏 = NSWT (roi)
• Prononciation complète: HTP DI NSWT ("hetep di nesout")
