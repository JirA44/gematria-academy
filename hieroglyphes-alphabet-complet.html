Manuel de lecture - Tous les signes essentiels pour déchiffrer les textes

📚 Comment utiliser cet alphabet

Cet alphabet contient tous les signes hiéroglyphiques essentiels organisés par catégorie. Pour chaque signe, vous trouverez :

  • Le symbole hiéroglyphique - L'apparence visuelle du signe
  • Le code Gardiner - Référence standard en égyptologie
  • La prononciation - Comment le lire à voix haute
  • Le nom français - Description de ce que représente le signe
  • L'explication - Comment et quand l'utiliser

Conseil : Utilisez la recherche pour trouver rapidement un signe, ou filtrez par catégorie pour apprendre progressivement.
