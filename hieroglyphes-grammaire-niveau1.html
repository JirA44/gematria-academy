📖 Leçon 1 : Le Sens de Lecture

Contrairement à notre écriture qui va toujours de gauche à droite, les hiéroglyphes peuvent se lire dans plusieurs directions.

🧭 Règle d'Or : Regardez où les êtres vivants "regardent"

✨ RÈGLE FONDAMENTALE

Les signes représentant des êtres vivants (humains, animaux, oiseaux) "regardent" toujours vers le DÉBUT de la ligne.

  • Si les signes regardent vers la gauche ← : on lit de gauche à droite
  • Si les signes regardent vers la droite → : on lit de droite à gauche
  • Si les signes regardent vers le bas ↓ : on lit de haut en bas

Exemple 1 : Lecture de gauche à droite

𓀀 𓇋𓈖𓂧 𓎛𓂋

Les signes regardent vers la gauche ← donc on lit : ind hr (salutations)

Exemple 2 : Lecture de droite à gauche

𓂋𓎛 𓂧𓈖𓇋 𓀀

Les signes regardent vers la droite → donc on lit : hr ind (salutations)

🔤 Leçon 2 : Les Phonogrammes Unilitères (L'Alphabet)

Les phonogrammes sont des signes qui représentent des sons. Les unilitères représentent une seule consonne.

🔍 Leçon 3 : Les Déterminatifs (Signes Muets)

Les déterminatifs sont des signes qui ne se prononcent PAS mais qui précisent la catégorie du mot (personne, animal, abstrait, etc.)

❓ Pourquoi des signes muets ?

🤔 Le Problème de l'Écriture Sans Voyelles

L'égyptien ancien n'écrit que les consonnes (comme l'hébreu). Imaginez en français si on écrivait seulement :

  • PR pourrait être : PRé, PReU, PouR, PaRi, PaiRe...
  • MR pourrait être : MèRe, MeR, MuR, éMiR...

Les déterminatifs aident à lever l'ambiguïté en indiquant le sens général du mot.

📊 Les Déterminatifs Essentiels

Déterminatif Catégorie Exemples
𓀀 Homme, masculin, "je" 𓇋𓈖𓂧 𓀀 (ind = salutation)
𓁐 Femme, féminin 𓆓𓏏 𓁐 (ḏt = corps féminin)
𓀭 Dieu, divinité 𓇋𓏠𓈖 𓀭 (imn = Amon)
𓏛 Abstrait, concept 𓅓𓏏 𓏛 (mt = mort [concept])
𓊖 Ville, lieu 𓉐𓂋 𓊖 (pr = maison [lieu])

Exemple : Même racine, sens différent

Regardons comment le déterminatif change le sens :

𓈖𓆑𓂋

NFR (sans déterminatif) = racine "bon/beau"

𓈖𓆑𓂋 𓀀

NFR + homme assis = jeune homme, bel homme

𓈖𓆑𓂋 𓏛

NFR + papyrus = beauté (concept abstrait)

⚠️ ATTENTION : Les déterminatifs sont TOUJOURS à la fin du mot et ne se prononcent JAMAIS !

🔊 Leçon 4 : Les Compléments Phonétiques

Les Égyptiens aimaient la redondance pour s'assurer qu'on prononce correctement. Ils ajoutaient souvent des unilitères pour "renforcer" la prononciation d'un signe bilitère ou trilitère.

Exemple : Le mot "nfr" (bon)

Plusieurs façons d'écrire le même mot :

𓄤

Version courte : un seul signe trilitère NFR

𓄤𓂋

Avec complément phonétique : NFR + R (pour confirmer le "r")

𓈖𓆑𓂋

Version complète : N + F + R (trois unilitères)

Ces trois versions se prononcent EXACTEMENT pareil : NFR

✨ RÈGLE D'OR DES COMPLÉMENTS

Quand vous voyez un signe bilitère ou trilitère suivi d'unilitères qui répètent une partie de sa prononciation, ne les prononcez qu'une seule fois !
