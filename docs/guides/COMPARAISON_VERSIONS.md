## ❓ Questions Fréquentes

### Puis-je utiliser les 3 versions en parallèle ?
Oui ! Elles sont **compatibles et synchronisées**. Votre progression est partagée.

### La version MEGA est-elle payante ?
**Non**, toutes les versions sont **100% gratuites** pour un usage personnel.

### Combien de temps pour maîtriser un système complet ?
- **Niveau Débutant** : 2-4 semaines
- **Niveau Intermédiaire** : 1-2 mois
- **Niveau Expert** : 3-6 mois
- **Certification complète** : 6-12 mois

### Les certifications sont-elles reconnues ?
Les certifications sont **reconnues dans la communauté ésotérique**.
Elles témoignent de votre maîtrise et peuvent être partagées (PDF, réseaux sociaux).

### Y a-t-il une application mobile ?
Pas encore, mais le site est **responsive** (fonctionne sur mobile/tablette).
Une **PWA** (Progressive Web App) est prévue en 2025.
