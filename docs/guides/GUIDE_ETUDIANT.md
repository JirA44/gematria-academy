# 📚 Guide de l'Étudiant - Système de Validation des Connaissances

## 🌟 Bienvenue dans votre parcours d'apprentissage !

Ce guide vous explique comment utiliser le système de validation pour progresser efficacement dans l'Académie des Sagesses Ésotériques.

---

## 🎯 Objectif du Système

Le système de validation vous permet de :
- ✅ **Vérifier** que vous avez bien compris chaque leçon
- 🔓 **Débloquer** progressivement de nouvelles leçons
- 📊 **Suivre** votre progression en temps réel
- 🏆 **Obtenir** des badges et certificats officiels
- 🎓 **Progresser** dans les niveaux jusqu'à devenir Maître Ésotérique

---

## 🚀 Comment Commencer

### Étape 1 : Étudiez la Leçon
1. Lisez attentivement le contenu de la leçon
2. Prenez des notes si nécessaire
3. Relisez les points importants
4. Assurez-vous de bien comprendre les concepts

### Étape 2 : Passez le Quiz
1. Cliquez sur le bouton **"🎓 Valider mes connaissances"**
2. Une fenêtre de quiz s'ouvrira avec 4 questions
3. Lisez chaque question attentivement
4. Choisissez la réponse qui vous semble correcte

### Étape 3 : Recevez le Feedback
- ✅ **Réponse correcte** : Vous verrez une coche verte + explication
- ❌ **Réponse incorrecte** : La bonne réponse sera affichée + explication
- Cliquez sur **"Question suivante →"** pour continuer

### Étape 4 : Consultez vos Résultats
À la fin du quiz, vous verrez :
- 📊 Votre **score** (sur 100%)
- 🎯 Votre **note** (A+, A, B+, B, C+, C, D)
- ✅ Nombre de **bonnes réponses**
- 📋 Possibilité de **revoir vos réponses**

### Étape 5 : Validez ou Réessayez
- **Score ≥ 80%** : 🎉 Leçon validée ! La suivante se débloque
- **Score < 80%** : 📚 Relisez la leçon et réessayez

## 🔒 Système de Déverrouillage

### Comment ça Marche
1. **Première leçon** : Toujours débloquée
2. **Leçons suivantes** : Se débloquent après validation de la précédente
3. **Score requis** : 80% minimum pour valider

### Badges Visuels sur les Cartes
```
🔒 VERROUILLÉ    →  Leçon non accessible encore
📖 EN COURS      →  Leçon disponible, non validée
✅ VALIDÉ        →  Leçon terminée avec succès
```

## 💡 Conseils pour Réussir

### 📖 Avant le Quiz
- ✅ Lisez **toute** la leçon avant de commencer
- ✅ Prenez des **notes** sur les points clés
- ✅ Relisez les **exemples** fournis
- ✅ Identifiez les **concepts principaux**

### 🎯 Pendant le Quiz
- ✅ Lisez **attentivement** chaque question
- ✅ Éliminez les réponses **évidemment fausses**
- ✅ Réfléchissez avant de cliquer
- ✅ Lisez les **explications** après chaque réponse

### 📚 Si Vous Échouez (< 80%)
- ✅ **Ne vous découragez pas** ! C'est normal
- ✅ Lisez les **explications** de chaque question
- ✅ Relisez la leçon en vous concentrant sur ce que vous avez raté
- ✅ **Réessayez** le quiz
- ✅ Utilisez le bouton **"Réviser et Réessayer"**

### 🎓 Pour Exceller
- ⭐ Visez **95%+** pour obtenir le badge Excellence
- ⭐ Validez **plusieurs leçons** rapidement pour le badge Sprint
- ⭐ Obtenez des **100%** pour le badge Perfectionniste
- ⭐ Complétez **tous les systèmes** pour les certificats

## 📥 Export de Progression

### Pourquoi Exporter ?
- 💾 **Sauvegarde** de votre progression
- 📊 **Analyse** de vos résultats
- 🔄 **Transfert** vers un autre appareil

### Comment Exporter
1. Ouvrez votre tableau de bord
2. Cliquez sur **"📥 Exporter ma Progression"**
3. Un fichier JSON sera téléchargé
4. Conservez-le en sécurité

Le fichier contient :
- Toutes vos validations
- Vos scores
- Vos badges
- Votre niveau et XP
- Date d'export

## ⚠️ Réinitialisation

### Attention !
La réinitialisation **supprime TOUTES vos données** :
- ❌ Toutes les validations
- ❌ Tous les scores
- ❌ Tous les badges
- ❌ Votre niveau et XP

### Quand Réinitialiser ?
- Vous voulez recommencer à zéro
- Vous avez fait des tests et voulez repartir propre
- Vous voulez un nouveau défi

### Comment Réinitialiser
1. Ouvrez votre tableau de bord
2. Cliquez sur **"🔄 Réinitialiser"**
3. **Confirmez 2 fois** (c'est irréversible !)
4. Exportez d'abord si vous voulez sauvegarder

## 🎓 Questions Fréquentes

### Q: Combien de questions par quiz ?
**R:** Chaque quiz contient **4 questions**.

### Q: Quel score pour valider ?
**R:** **80% minimum** (3 bonnes réponses sur 4).

### Q: Puis-je réessayer un quiz ?
**R:** **Oui**, autant de fois que nécessaire jusqu'à obtenir 80%+.

### Q: Les quiz changent-ils à chaque fois ?
**R:** Non, ce sont les mêmes questions. L'objectif est d'**apprendre**, pas de tester votre chance.

### Q: Puis-je revenir en arrière ?
**R:** Oui, vous pouvez **refaire** les quiz validés pour améliorer votre score.

### Q: Les données sont-elles sauvegardées ?
**R:** Oui, dans votre navigateur (localStorage). Exportez pour sauvegarder ailleurs.

### Q: Dois-je créer un compte ?
**R:** Non, tout fonctionne en local. Vos données restent sur votre appareil.

### Q: Comment obtenir un certificat ?
**R:** Validez **toutes les leçons** d'un système (ex: toutes les Runes) puis téléchargez-le depuis le tableau de bord.
