## 🎨 Personnalisation

### Changer le Thème
Le thème par défaut utilise des dégradés violet/magenta. Pour personnaliser :

1. Ouvrez `style.css`
2. Modifiez les variables CSS :
```css
:root {
    --primary-color: #8a2be2;     /* Violet principal */
    --secondary-color: #ff00ff;   /* Magenta */
    --accent-color: #6a0dad;      /* Violet foncé */
}
```

### Ajouter Vos Propres Contenus
Chaque section utilise une structure modulaire. Pour ajouter une leçon :

1. Créez un nouveau fichier HTML (ex: `tarot-lame-01-bateleur.html`)
2. Utilisez le template de leçon existant
3. Ajoutez le lien dans la page de niveau correspondante
