🎨 Générateur d'Icônes PWA

Cliquez sur "Télécharger tout" pour obtenir toutes les icônes nécessaires pour votre PWA

📋 Instructions d'installation :

  1. Téléchargez toutes les icônes en cliquant sur le bouton ci-dessus
  2. Placez les fichiers dans le dossier /assets/icons/
  3. Les icônes seront automatiquement détectées par le manifest.json
  4. Testez votre PWA avec Lighthouse pour vérifier l'installation
