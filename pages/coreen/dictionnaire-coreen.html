📐 Grammaire Coréenne

1. Structure de la Phrase

Le coréen suit l'ordre Sujet - Objet - Verbe (SOV). Le verbe est toujours à la fin.

저는 밥을 먹어요
jeoneun bab-eul meogeoyo
Je + (sujet) + riz + (objet) + mange
= "Je mange du riz"
2. Les Particules

Les particules indiquent la fonction grammaticale des mots.

Particule Fonction Utilisation Exemple
은/는 Sujet (thème) 은 après consonne, 는 après voyelle 저는 (jeoneun) = moi, je...
이/가 Sujet (nouveau) 이 après consonne, 가 après voyelle 고양이가 (goyang-iga) = le chat
을/를 Objet direct 을 après consonne, 를 après voyelle 밥을 (bab-eul) = le riz (obj)
에 Lieu, temps Direction, moment 학교에 (hakgyo-e) = à l'école
에서 Lieu (action) Où se passe l'action 학교에서 (hakgyo-eseo) = à l'école
의 Possession De (possessif) 나의 (na-ui) = mon, ma
3. Les Niveaux de Politesse

Le coréen a plusieurs niveaux de politesse exprimés par les terminaisons verbales.

Niveau Terminaison Exemple (먹다 = manger) Usage
Très formel (합쇼체) -ㅂ니다/-습니다 먹습니다 (meokseumnida) Discours, présentations
Poli (해요체) -아요/-어요 먹어요 (meogeoyo) Conversations polies
Informel (반말) -아/-어 먹어 (meogeo) Amis proches, enfants
4. Conjugaison de Base

Les verbes coréens se conjuguent en remplaçant 다 par la terminaison appropriée.

가다 (gada) = aller

Présent poli : 가요 (gayo) - Je vais
Présent formel : 갑니다 (gamnida) - Je vais
Passé poli : 갔어요 (gasseoyo) - Je suis allé(e)
Futur poli : 갈 거예요 (gal geoyeyo) - J'irai
5. La Négation

Deux façons de former la négation :

Avec 안 (an) devant le verbe :
먹어요 → 안 먹어요 (an meogeoyo) = Je ne mange pas

Avec -지 않다 (-ji anta) après le radical :
먹다 → 먹지 않아요 (meokji anayo) = Je ne mange pas
6. Les Questions

En coréen, l'intonation monte à la fin pour les questions. La structure reste la même.

Affirmation : 밥을 먹어요. (Bab-eul meogeoyo.) = Je mange du riz.
Question : 밥을 먹어요? (Bab-eul meogeoyo?) = Tu manges du riz ?

Mots interrogatifs :
뭐 (mwo) = quoi
누구 (nugu) = qui
어디 (eodi) = où
언제 (eonje) = quand
왜 (wae) = pourquoi
어떻게 (eotteoke) = comment

Le Hangeul - Un Alphabet Scientifique

Le Hangeul a été créé en 1443 par le roi Sejong le Grand pour permettre à tous les Coréens de lire et écrire. Les consonnes représentent la position de la langue et les voyelles sont basées sur les principes du yin et yang. C'est considéré comme l'un des systèmes d'écriture les plus scientifiques au monde.
