📐 Grammaire Égyptienne

1. Structure de la Phrase

L'égyptien ancien suit généralement l'ordre Verbe - Sujet - Objet (VSO), contrairement au français (SVO).

𓏙 𓊹 𓋹 → dỉ nṯr ꜥnḫ
Donne + dieu + vie = "Le dieu donne la vie"
2. Les Déterminatifs

Les déterminatifs sont des signes muets placés à la fin des mots pour indiquer leur catégorie sémantique.

Déterminatif Catégorie Exemple
𓀀 Homme, personne masculine 𓊃 𓀀 = s "homme"
𓁐 Femme, personne féminine 𓊃𓏏 𓁐 = st "femme"
𓊹 Divinité 𓇋𓏠𓈖 𓊹 = Ỉmn "Amon"
𓏏𓏤 Abstraction 𓄤 𓏏𓏤 = nfrt "beauté"
𓃀𓃀 Mouvement, marche 𓇋𓇋 𓃀𓃀 = ỉỉ "venir"
3. Genre et Nombre

Genre : Masculin (sans marque) / Féminin (suffixe 𓏏 -t)

Nombre : Singulier / Duel (suffixe 𓏭 -wy/-ty) / Pluriel (suffixe 𓏥 -w)

𓊹 nṯr = "dieu" (masc. sing.)
𓊹𓏏 nṯrt = "déesse" (fém. sing.)
𓊹𓏥 nṯrw = "dieux" (masc. plur.)
4. Les Pronoms Suffixes

Les pronoms se placent après le nom ou le verbe.

Personne Suffixe Exemple
Je / Mon 𓇋 -ỉ / =ỉ pr=ỉ "ma maison"
Tu (masc.) / Ton 𓎡 -k / =k pr=k "ta maison"
Tu (fém.) / Ton 𓏏 -t / =t pr=t "ta maison"
Il / Son 𓆑 -f / =f pr=f "sa maison"
Elle / Sa 𓊃 -s / =s pr=s "sa maison"
Nous / Notre 𓈖 -n / =n pr=n "notre maison"
Ils / Leur 𓊃𓈖 -sn / =sn pr=sn "leur maison"
5. La Négation

La négation se forme avec 𓈖 n devant le verbe, ou 𓈖𓈖 nn pour une négation plus forte.

𓈖 𓅓𓂝𓂝 = n mꜣꜣ = "il ne voit pas"
𓈖𓈖 𓂝𓎡 = nn ꜥk = "rien n'entre"
6. La Possession (Génitif)

Le génitif direct : le possesseur suit directement le possédé.

Le génitif indirect : utilise 𓈖 n(y) "de".

𓉐 𓊹 = pr nṯr = "maison du dieu" (direct)
𓊃𓏏 𓈖 𓇓 = sꜣt n(y)t nsw = "fille du roi" (indirect)

📜 Note sur l'écriture

L'égyptien hiéroglyphique ne note généralement pas les voyelles. Les égyptologues ajoutent un "e" entre les consonnes pour faciliter la prononciation. Ainsi "nfr" se prononce conventionnellement "néfer".
