Fallacies, raisonnement formel, logique propositionnelle et connecteurs logiques.

🚨 1. Les Sophismes (Fallacies) — Erreurs de raisonnement courantes

Un sophisme est un argument qui semble valide mais qui est logiquement incorrect. Les reconnaître est une compétence défensive essentielle.

Ad Hominem

Attaquer la personne plutôt que son argument.

"Tu dis que le tabac est nocif, mais tu fumes !"

Homme de Paille

Déformer l'argument adverse pour le réfuter facilement.

"Vous voulez réduire le budget militaire ? Vous voulez qu'on soit sans défense !"

Faux Dilemme

Présenter deux options comme les seules possibles.

"Soit tu es avec nous, soit tu es contre nous."

Appel à l'Autorité

Utiliser l'opinion d'une autorité hors de son domaine.

"Ce footballeur dit que ce médicament est efficace."

Pente Glissante

Supposer qu'une action entraîne inévitablement une catastrophe.

"Si on autorise X, bientôt tout sera permis !"

Post Hoc Ergo Propter Hoc

Confondre succession et causalité.

"J'ai mis ce fétiche, puis j'ai gagné. Donc le fétiche m'a fait gagner."

Appel à la Popularité

Conclure qu'une chose est vraie parce qu'elle est largement crue.

"Des milliards de gens y croient, ça doit être vrai."

Généralisation Hâtive

Tirer une règle générale d'un trop petit échantillon.

"J'ai rencontré deux Français arrogants, ils sont tous pareils."

💡 Pour identifier un sophisme : isolez la structure de l'argument, puis demandez-vous si la conclusion suit nécessairement des prémisses.

⚙️ 2. Logique Propositionnelle — Les Connecteurs Logiques

La logique propositionnelle manipule des propositions (P, Q, R…) reliées par des connecteurs. Chaque proposition est soit vraie (V) soit fausse (F).

Les 5 connecteurs de base

  • ¬P (NON) — Négation. Si P est V, ¬P est F.
  • P ∧ Q (ET) — Conjonction. Vrai seulement si P ET Q sont vrais.
  • P ∨ Q (OU) — Disjonction. Vrai si au moins l'un est vrai.
  • P → Q (SI…ALORS) — Implication. Faux seulement si P est vrai et Q est faux.
  • P ↔ Q (SI ET SEULEMENT SI) — Équivalence. Vrai si P et Q ont la même valeur.

Table de vérité — Implication (P → Q)

PQP → QInterprétation
VVVPromesse tenue
VFFPromesse brisée ← seul cas faux
FVVPrémisse fausse → tout est "valide"
FFVPrémisse fausse → tout est "valide"
Lecture concrète de P → Q P = "Il pleut" | Q = "Je prends mon parapluie"
P → Q = "S'il pleut, je prends mon parapluie."

Cas FAUX : il pleut (P=V) mais je n'ai pas de parapluie (Q=F) → j'ai menti.

🔗 3. Formes Valides Fondamentales

Modus Ponens (affirmer la prémisse)

Structure P1 : P → Q (Si P alors Q)
P2 : P (P est vrai)
C : Q (Donc Q est vrai) ✅
Exemple concret P1 : S'il pleut, la route est mouillée.
P2 : Il pleut.
C : Donc, la route est mouillée.

Modus Tollens (nier la conséquence)

Structure P1 : P → Q (Si P alors Q)
P2 : ¬Q (Q est faux)
C : ¬P (Donc P est faux) ✅
Exemple concret P1 : Si c'est un mammifère, ça allaite ses petits.
P2 : Ça n'allaite pas ses petits.
C : Ce n'est pas un mammifère.

Erreur fréquente : Affirmer le conséquent (INVALIDE)

Structure invalide ❌ P1 : P → Q
P2 : Q ← on affirme la conséquence
C : P ← INVALIDE !
Exemple d'erreur P1 : S'il pleut, le sol est mouillé.
P2 : Le sol est mouillé.
C : Donc il pleut. ← FAUX (quelqu'un a pu arroser !)

🌀 4. Paradoxes Logiques Classiques

Les paradoxes montrent les limites de la logique formelle et ont conduit à des révolutions mathématiques.

  • Paradoxe du Menteur — "Cette phrase est fausse."
    Si elle est vraie, elle est fausse. Si elle est fausse, elle est vraie. → Auto-référence impossible.
  • Paradoxe du Barbier (Russell) — "Le barbier rase tous ceux qui ne se rasent pas eux-mêmes. Qui rase le barbier ?"
    A conduit Russell à reformuler les fondements des mathématiques (théorie des types).
  • Paradoxe de Zénon (Achille et la Tortue) — Achille ne peut jamais rattraper la tortue si elle a de l'avance.
    Résolu par le calcul infinitésimal : somme infinie convergente.
  • Paradoxe du Sorite (tas de sable) — Enlever un grain d'un tas ne détruit pas le tas. Alors combien de grains font un tas ?
    Illustre le problème de la vagueness (logique floue).
