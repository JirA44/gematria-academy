🌱 Logique Formelle — Niveau Débutant

Maîtrisez les bases du raisonnement : syllogismes, déduction, induction et abduction.

🧩 1. Qu'est-ce que la logique ?

La logique est l'étude des règles du raisonnement valide. Elle permet de distinguer les arguments corrects des arguments fallacieux, indépendamment de leur contenu.

  • Argument : Ensemble de propositions dont l'une (la conclusion) est censée découler des autres (les prémisses).
  • Validité : Un argument est valide si la conclusion suit nécessairement des prémisses.
  • Solidité : Valide + prémisses vraies = argument solide.
  • Proposition : Énoncé pouvant être vrai ou faux (ex : "Il pleut").
💡 Un argument valide avec de fausses prémisses peut mener à une fausse conclusion. La logique garantit la forme, pas le contenu.

🏛️ 2. Le Syllogisme (Aristote ~350 av. J.-C.)

Le syllogisme est la forme la plus ancienne de raisonnement formel. Il comporte trois parties :

  • Prémisse majeure : Règle générale
  • Prémisse mineure : Cas particulier
  • Conclusion : Ce qui en découle
Syllogisme classique P1 (majeure) : Tous les humains sont mortels.
P2 (mineure) : Socrate est un humain.
C (conclusion): Donc, Socrate est mortel.
Syllogisme avec animaux P1 : Tous les mammifères sont des vertébrés.
P2 : Le dauphin est un mammifère.
C : Donc, le dauphin est un vertébré.

Syllogismes invalides (pièges courants)

Erreur de la conversion (INVALIDE) P1 : Tous les chiens sont des animaux.
P2 : Tous les chats sont des animaux.
C : Donc, tous les chiens sont des chats. ❌

→ Erreur : "Tous les A sont B" ne signifie pas "Tous les B sont A".

🎯 3. Le Raisonnement Déductif

La déduction va du général au particulier. Si les prémisses sont vraies et le raisonnement valide, la conclusion est certaine.

+
Cas particulier
P2
→
Conclusion certaine

Exemple mathématique P1 : Si un nombre est divisible par 4, il est divisible par 2.
P2 : 12 est divisible par 4.
C : Donc, 12 est divisible par 2. ✅ (Certain)
  • Avantage : Certitude absolue si les prémisses sont vraies.
  • Limite : On ne découvre rien de "nouveau" — la conclusion est contenue dans les prémisses.
  • Usage : Mathématiques, droit, programmation.

🔬 4. Le Raisonnement Inductif

L'induction va du particulier au général. Elle génère des règles générales à partir d'observations. La conclusion est probable, jamais certaine.

Observations
faits
→
Règle générale probable
C
Exemple classique Observation 1 : Ce cygne est blanc.
Observation 2 : Ce cygne est blanc.
Observation 3 : Ce cygne est blanc.
…
Conclusion : Tous les cygnes sont blancs. (Probable mais FAUX — il existe des cygnes noirs en Australie)
Usage scientifique Obs. 1-100 : Ce médicament réduit la fièvre chez 100 patients.
Conclusion : Ce médicament est efficace contre la fièvre. (Probable, à confirmer)
  • Avantage : Permet de formuler des hypothèses et de progresser.
  • Limite : Un seul contre-exemple suffit à réfuter la règle (Karl Popper : "falsifiabilité").
  • Usage : Science empirique, épidémiologie, machine learning.
💡 La science fonctionne par induction + falsification : on cherche des contre-exemples pour tester les théories.

🕵️ 5. Le Raisonnement Abductif

L'abduction cherche la meilleure explication possible à un fait observé. C'est le raisonnement du détective.

Fait observé
surprenant
→
Meilleure explication
hypothèse
Sherlock Holmes style Observation : Le sol est mouillé.
Explication 1 : Il a plu. (probable)
Explication 2 : Un tuyau a éclaté. (possible)
Explication 3 : Quelqu'un a renversé de l'eau. (moins probable)
Abduction : La meilleure explication → il a plu.
  • Utilisé par : Médecins (diagnostic), détectives, scientifiques.
  • Avantage : Permet d'agir rapidement avec des informations incomplètes.
  • Limite : La "meilleure" explication peut être fausse.

📊 6. Comparatif : Déduction / Induction / Abduction

Type Direction Certitude Usage
Déduction Général → Particulier Certaine (si prémisses vraies) Maths, droit
Induction Particulier → Général Probable (jamais certaine) Science, IA
Abduction Observation → Hypothèse Plausible (meilleure explication) Diagnostic, investigation
