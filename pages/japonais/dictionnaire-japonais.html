📐 Grammaire Japonaise

1. Structure de la Phrase

Le japonais suit l'ordre Sujet - Objet - Verbe (SOV). Le verbe est toujours à la fin.

私は寿司を食べます
watashi wa sushi wo tabemasu
Je + (thème) + sushi + (objet) + mange
= "Je mange des sushis"
2. Les Particules

Les particules indiquent la fonction grammaticale des mots.

Particule Fonction Exemple
は (wa) Thème de la phrase 私は (watashi wa) = Moi, je...
が (ga) Sujet (nouveau/emphase) 猫がいます (neko ga imasu) = Il y a un chat
を (wo) Objet direct 本を読む (hon wo yomu) = lire un livre
に (ni) Direction, temps, cible 学校に行く (gakkou ni iku) = aller à l'école
で (de) Lieu d'action, moyen 車で行く (kuruma de iku) = aller en voiture
の (no) Possession 私の本 (watashi no hon) = mon livre
と (to) Et, avec 友達と (tomodachi to) = avec un ami
か (ka) Question 食べますか (tabemasu ka) = Mangez-vous?
3. Les Niveaux de Politesse (敬語 Keigo)
Niveau Style Exemple (食べる = manger) Usage
Très poli (尊敬語) Honorifique 召し上がる (meshiagaru) Client, supérieur
Poli (丁寧語) -ます/-です 食べます (tabemasu) Standard, inconnus
Informel Forme dictionnaire 食べる (taberu) Amis, famille
4. Conjugaison des Verbes

Il existe 3 groupes de verbes en japonais :

Groupe Caractéristique Exemple Forme -ます
Groupe 1 (五段) Terminaison en -u 書く (kaku) - écrire 書きます (kakimasu)
Groupe 2 (一段) Terminaison -iru/-eru 食べる (taberu) - manger 食べます (tabemasu)
Groupe 3 (不規則) Irréguliers する (suru), 来る (kuru) します, 来ます
5. Les Temps
Présent/Futur :
食べます (tabemasu) = Je mange / Je mangerai

Passé :
食べました (tabemashita) = J'ai mangé

Progressif :
食べています (tabete imasu) = Je suis en train de manger
6. La Négation
Présent négatif poli :
食べます → 食べません (tabemasen) = Je ne mange pas

Passé négatif poli :
食べました → 食べませんでした (tabemasen deshita) = Je n'ai pas mangé

Forme informelle :
食べる → 食べない (tabenai) = Ne pas manger
7. Les Adjectifs

Deux types d'adjectifs en japonais :

Type Forme Exemple Avec nom
い-adjectifs Terminent en い 大きい (ookii) - grand 大きい本 (ookii hon)
な-adjectifs Prennent な devant nom 静か (shizuka) - calme 静かな部屋 (shizuka na heya)

Les Trois Systèmes d'Écriture

Le japonais utilise trois systèmes d'écriture simultanément :
Hiragana (ひらがな) : pour les mots japonais, grammaire
Katakana (カタカナ) : pour les mots étrangers, onomatopées
Kanji (漢字) : caractères chinois pour les concepts
