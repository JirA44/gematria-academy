🔢 Les Différents Systèmes de Gematria

Il existe plusieurs méthodes de calcul en gematria, chacune révélant des insights différents :

1. Gematria Absolue (Mispar Hechrachi)

C'est le système simple que vous connaissez : א=1, ב=2, etc.

2. Gematria Ordinale (Mispar Siduri)

Chaque lettre vaut sa position dans l'alphabet : א=1, ב=2... ת=22

Lettre Absolue Ordinale
י 10 10
ק 100 19
ת 400 22

3. Gematria Réduite (Mispar Katan)

On réduit chaque nombre à un chiffre : 10→1, 20→2, 100→1, etc.

Exemple : ש (Shin) = 300 → 3+0+0 = 3

4. Gematria "AtBash" (אתב״ש)

Système de substitution inversé : א↔ת, ב↔ש, ג↔ר, etc.

La première lettre devient la dernière, la deuxième devient l'avant-dernière...

✍️ Exercice : AtBash

Convertissez le mot בבל (Babel) en AtBash :

  • ב → ש
  • ב → ש
  • ל → כ

Résultat : ששכ - qui se lit "Sheshakh", un nom codé de Babylone dans Jérémie 25:26 !

📚 Notarikon et Temurah

Deux autres techniques kabbalistiques d'analyse des textes :

Notarikon (נוטריקון)

Technique d'acronymes où chaque lettre d'un mot représente un mot entier.

Exemple célèbre : בראשית (Bereshit - "Au commencement")

Chaque lettre est l'initiale d'un mot :

  • ב - בראשית (Bereshit - Au commencement)
  • ר - ראה (Raah - Il vit)
  • א - אלהים (Elohim - Dieu)
  • ש - שיקבלו (Sheyikabelu - Qu'ils acceptent)
  • י - ישראל (Israël - Israël)
  • ת - תורה (Torah - Torah)

"Au commencement, Dieu vit qu'Israël accepterait la Torah"

Temurah (תמורה)

Permutations et substitutions de lettres selon des règles fixes.

Exemples de systèmes Temurah :

  • AtBash (déjà vu) : א↔ת, ב↔ש...
  • Albam : א↔ל, ב↔מ, ג↔נ...
  • Atbah : א↔ט, ב↔ח, ג↔ז...

📖 Gematria dans la Torah

Exemples de correspondances révélées par la gematria dans les textes sacrés :

Exemple 1 : Jacob et Esaü

יעקב

Jacob (יעקב) = 10+70+100+2 = 182

עשו

Esaü (עשו) = 70+300+6 = 376

376 - 182 = 194 = צדקה (Tsedakah - Charité)

Enseignement : La charité comble la différence entre les frères.

Exemple 2 : Le serpent et le Messie

נחש

Serpent (נחש) = 50+8+300 = 358

משיח

Messie (משיח) = 40+300+10+8 = 358

Enseignement : Le Messie vient vaincre le serpent primordial.

✍️ Exercice pratique : Analyse comparative

Calculez ces mots avec les trois systèmes (Absolue, Ordinale, Réduite) :

  1. תורה (Torah - Loi) : ת+ו+ר+ה
  2. חכמה (Hokhma - Sagesse) : ח+כ+מ+ה
  3. שלום (Shalom - Paix) : ש+ל+ו+ם

Cherchez des correspondances entre les totaux !

Indice :

Torah en gematria absolue = 611, soit le nombre de commandements donnés par Moïse (613 - 2 donnés directement par Dieu au Sinaï) !

✍️ Projet : Votre Nom en Hébreu

  1. Translittérez votre prénom en lettres hébraïques
  2. Calculez sa gematria absolue
  3. Trouvez un mot hébreu avec la même valeur
  4. Méditez sur cette correspondance personnelle

Exemple : "David" = דוד = 4+6+4 = 14 = יד (Yad - Main)
